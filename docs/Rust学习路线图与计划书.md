## 2. 第一阶段：基础入门（2-3周）

### 2.1 目标
掌握 Rust 基础语法，能够编写简单的程序。

### 2.2 学习计划

| 天数 | 主题 | 学习内容 | 实践任务 |
|------|------|----------|----------|
| 1-2 | 环境搭建 | 安装 Rust、Cargo、IDE 配置 | 创建第一个项目，运行 "Hello, World!" |
| 3-4 | 变量与数据类型 | 可变性、标量类型、复合类型 | 编写类型转换和变量shadowing示例 |
| 5-6 | 流程控制 | if/else、loop、while、for | 实现猜数字游戏的基础逻辑 |
| 7-8 | 函数 | 参数、返回值、语句与表达式 | 重构代码，提取函数 |
| 9-10 | 注释与文档 | 行注释、块注释、文档注释 | 为代码添加文档 |
| 11-12 | 模块系统 | mod、use、pub、crate | 组织多文件项目 |
| 13-14 | 复习与练习 | 完成所有基础练习题 | 完成一个小工具程序 |

### 2.3 关键知识点

```rust
// 1. 变量与可变性
let x = 5;           // 不可变
let mut y = 10;      // 可变
const MAX: u32 = 100; // 常量

// 2. 数据类型
let guess: u32 = "42".parse().expect("Not a number!");
let tup: (i32, f64, u8) = (500, 6.4, 1);
let arr: [i32; 5] = [1, 2, 3, 4, 5];

// 3. 函数
fn add(a: i32, b: i32) -> i32 {
    a + b  // 注意：没有分号，这是表达式
}

// 4. 流程控制
let number = if condition { 5 } else { 6 };

// 5. 循环
for i in 1..=10 {
    println!("{}", i);
}
```

### 2.4 阶段检查点
- [ ] 能够独立创建和运行 Rust 项目
- [ ] 理解可变性与不可变性的区别
- [ ] 掌握函数的定义和调用
- [ ] 理解模块系统的基本用法

## 3. 第二阶段：核心概念（3-4周）

### 3.1 目标
深入理解 Rust 的核心概念，特别是所有权系统。

### 3.2 学习计划

| 天数 | 主题 | 学习内容 | 实践任务 |
|------|------|----------|----------|
| 1-3 | 所有权系统 | 所有权规则、move、Copy trait | 实现字符串处理函数 |
| 4-6 | 引用与借用 ✅ | 引用规则、可变引用、悬垂引用 | 阅读已有文档，完成练习 |
| 7-9 | Slice 类型 | 字符串 slice、数组 slice | 实现单词查找函数 |
| 10-12 | 结构体 ✅ | 定义、实例化、方法 | 阅读已有文档，实现面向对象示例 |
| 13-15 | 枚举与模式匹配 ✅ | Option、Result、match | 阅读已有文档，实现状态机 |
| 16-18 | 泛型、Trait、生命周期 ✅ | 综合应用 | 阅读已有文档，实现通用容器 |
| 19-21 | 错误处理 ✅ | Result、?运算符、自定义错误 | 阅读已有文档，完善错误处理 |

> ✅ 标记表示你已有相关文档，重点复习和实践

### 3.3 核心概念思维导图

```
                    所有权系统
                        │
        ┌───────────────┼───────────────┐
        ▼               ▼               ▼
    所有权规则       引用与借用         生命周期
        │               │               │
   ┌────┴────┐     ┌────┴────┐     ┌────┴────┐
   │ 每个值   │     │ &T 借用  │     │ 编译时   │
   │ 一个所有者│     │ &mut T   │     │ 检查    │
   │ 唯一所有者│     │ 可变借用 │     │ 引用有效性│
   └────┬────┘     └────┬────┘     └────┬────┘
        │               │               │
        └───────────────┴───────────────┘
                        │
                        ▼
                  内存安全保证
                  （无 GC、无泄漏）
```

### 3.4 阶段检查点
- [ ] 完全理解所有权三规则
- [ ] 能够在复杂场景下正确使用引用
- [ ] 熟练使用 match 和模式匹配
- [ ] 能够使用泛型和 Trait 编写通用代码
- [ ] 理解生命周期标注的含义

## 4. 第三阶段：进阶特性（3-4周）

### 4.1 目标
掌握 Rust 的高级特性，能够编写生产级代码。

### 4.2 学习计划

| 天数 | 主题 | 学习内容 | 实践任务 |
|------|------|----------|----------|
| 1-3 | 集合类型 | Vec、HashMap、HashSet、BTreeMap | 实现数据统计程序 |
| 4-6 | 迭代器与闭包 | Iterator trait、闭包捕获、Fn trait | 实现数据处理管道 |
| 7-9 | 智能指针 | Box、Rc、RefCell、Arc、Mutex | 实现链表、共享状态 |
| 10-12 | 并发编程 | 线程、消息传递、共享状态 | 实现多线程下载器 |
| 13-15 | 异步编程 | async/await、Future、Tokio | 实现异步 HTTP 客户端 |
| 16-18 | 宏编程 | 声明宏、过程宏 | 编写自定义宏 |
| 19-21 |  unsafe Rust | 裸指针、extern、unsafe 块 | 了解 FFI 基础 |

### 4.3 进阶概念速览

```rust
// 1. 智能指针
use std::rc::Rc;
use std::cell::RefCell;

let shared = Rc::new(RefCell::new(vec![1, 2, 3]));
shared.borrow_mut().push(4);

// 2. 并发
use std::thread;
use std::sync::mpsc;

let (tx, rx) = mpsc::channel();
thread::spawn(move || {
    tx.send("Hello from thread").unwrap();
});

// 3. 异步
use tokio::time::{sleep, Duration};

async fn fetch_data() -> String {
    sleep(Duration::from_secs(1)).await;
    "Data".to_string()
}

#[tokio::main]
async fn main() {
    let data = fetch_data().await;
}
```

### 4.4 阶段检查点
- [ ] 熟练使用标准库集合
- [ ] 能够编写高效的迭代器链
- [ ] 理解并能正确使用智能指针
- [ ] 能够编写多线程程序
- [ ] 理解异步编程模型

## 总结

Rust 学习曲线较陡，但一旦掌握所有权系统，后面的学习就会顺利很多。建议：

1. **不要跳过所有权**：这是 Rust 的核心，花时间彻底理解
2. **多写代码**：光看文档不够，每天至少写 30 分钟代码
3. **阅读标准库源码**：学习优秀代码的写法
4. **参与社区**：遇到问题及时求助，帮助他人也能加深理解
5. **保持耐心**：掌握 Rust 需要时间，不要因为初期困难而放弃

**预计总学习时间**：8-12 周达到中级水平，持续实践达到高级水平。

祝学习愉快！
